# EPSS Master

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

EPSS Master is a tool for working with keys, certificates and files for sign and design.

## Features

- EPSS management: Sign, de-sign ...

## Requirements

- [FreeBSD.js >= 13.x](https://www.freebsd.org/where/)
- [Node.js >= 16.x](https://nodejs.org/en/download/)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/.../epss-master.git
   ```
